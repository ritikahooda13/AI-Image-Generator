AI Image Generator

A modern, full-stack Web Application built as part of the Python & Data Science Internship Project Phase 1 at Sqrock IT Solutions. This application leverages a Python backend with Flask to handle user prompts and generate high-quality AI images dynamically.

## 🚀 Features
* **Attractive UI:** Clean, modern, and fully responsive layout designed with Tailwind CSS.
* **Text Prompt Input:** Interactive input area allowing users to type custom descriptions.
* **AI Image Generation & Preview:** Fast backend pipeline processing to display the generated image instantly.
* **Instant Download Module:** A seamless mechanism to save the generated image directly to the local system.

## 🛠️ Tech Stack
* **Backend:** Python, Flask
* **Frontend:** HTML5, Tailwind CSS, JavaScript
* **Communication:** JSON-based Asynchronous Fetch API
